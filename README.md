# Калькулятор стоимости строительства — система обновления цен

Эта папка содержит всё необходимое, чтобы редактировать цены калькулятора в Excel и одной командой обновлять готовый HTML-файл для сайта.

## Состав

| Файл | Что делает |
|---|---|
| **prices.xlsx** | ✏️ Здесь вы редактируете цены. Жёлтые ячейки = редактируемые. |
| **calculator-template.html** | Шаблон калькулятора. Не трогать без необходимости. |
| **update_calculator.py** | Скрипт, который читает Excel и собирает готовый HTML. |
| **update.command** | 🖱️ Двойной клик — обновить один раз. |
| **watch.command** | 🖱️ Двойной клик — авто-обновление при каждом сохранении Excel. |
| **calculator.html** | ⬆️ Готовый файл для загрузки на сайт. **Пересоздаётся при каждом обновлении.** |

## Первый запуск на Mac

1. **Распакуйте архив в любую папку** (например, `~/Documents/calc-system/`).

2. **Установите Python 3**, если ещё нет (проверьте командой `python3 --version` в Терминале):
   ```bash
   # Если нет Homebrew, сначала:
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   # Затем:
   brew install python
   ```

3. **Разрешите запуск .command-файлов** — macOS блокирует их при первом запуске:
   - правый клик на `update.command` → «Открыть» → подтвердить «Открыть» в диалоге;
   - то же для `watch.command`.
   - Дальше работают по двойному клику.

4. **Сделайте файлы исполняемыми** (один раз, в Терминале):
   ```bash
   cd ~/Documents/calc-system  # путь к вашей папке
   chmod +x update.command watch.command
   ```

## Как пользоваться

### Изменить цену (типовой сценарий)

1. Откройте **prices.xlsx** в Excel (или Numbers — тоже работает).
2. Поправьте цену в жёлтой ячейке. Например, цену на «Клеёный брус» с 49 800 на 52 000.
3. **Сохраните** (Cmd+S).
4. Двойной клик по **update.command**.
5. Файл **calculator.html** обновился — заливайте на сайт.

### Режим авто-обновления

Если активно правите цены и хотите видеть результат в браузере без ручного запуска:

1. Двойной клик по **watch.command** → откроется окно Терминала.
2. Сохраните `prices.xlsx` после правок → `calculator.html` пересоберётся автоматически в течение 1 сек.
3. Обновите страницу в браузере (Cmd+R).
4. Когда закончили — закройте окно Терминала или нажмите Ctrl+C.

### Добавить новую опцию (например, ещё один фундамент)

1. На листе **«Опции»** добавьте новую строку:
   - **Категория:** `Фундамент`
   - **Ключ:** `winged_piles` (латиницей, без пробелов, уникальный)
   - **Название:** «Винтовые сваи»
   - **Цена ₽/м²:** 5500
   - **Активен:** 1
2. Сохраните Excel → запустите `update.command`.
3. Опция появится в выпадающем списке калькулятора.

### Скрыть опцию временно

Не удаляйте строку — поставьте **0** в колонке «Активен». Цена сохранится, опция просто не покажется клиенту.

### Изменить контакты или заголовок

Лист **«Контакты»** — редактируйте телефон, заголовок, WhatsApp-номер. После сохранения — запустите `update.command`.

### Настроить расчётные коэффициенты

Лист **«Коэффициенты»** — там этажные множители (1/1.5/2 этажа), надбавка на проектирование (7%), фиксированная цена ПОС и параметры расчёта площади фасада.

## Что куда заливать на сайте

Для сайта нужен только один файл — **calculator.html**. Залейте его в нужный раздел Bitrix / включаемую область / iframe. Все цены вшиты внутрь файла, никаких внешних зависимостей нет.

## Если что-то пошло не так

- **Ошибка «Не найден файл prices.xlsx»** → файлы должны лежать в одной папке.
- **Ошибка «Нет ни одной активной опции»** → проверьте, что в колонке «Активен» хотя бы для одной опции стоит 1.
- **Скрипт молча падает** → откройте Терминал, перейдите в папку (`cd ~/путь/calc-system`) и запустите вручную: `python3 update_calculator.py` — увидите детальную ошибку.
- **macOS говорит «файл не может быть открыт, разработчик не идентифицирован»** → правый клик → «Открыть» (один раз, потом будет работать по двойному клику).

## Структура prices.xlsx

### Лист «Опции»
Главная таблица с опциями калькулятора. **Категорию и Ключ менять не нужно** — это технические поля. **Цена** и **Активен** — редактируемые.

### Лист «Коэффициенты»
- Этажные множители (1, 1.5, 2 этажа)
- Надбавка на проектирование (0.07 = 7%)
- Фиксированная цена ПОС
- Параметры для расчёта площади фасада

### Лист «Контакты»
Телефон, WhatsApp, заголовок калькулятора, текст кнопки ПОС.

## Как работает связь Excel ↔ калькулятор

```
prices.xlsx (вы редактируете)
       ↓
update_calculator.py (скрипт читает Excel)
       ↓
calculator-template.html (шаблон с маркером {{PRICES_INJECTION}})
       ↓
calculator.html (готовый файл — все цены вшиты в JS-переменную PRICES внутри HTML)
       ↓
сайт (заливаете один файл, никаких внешних запросов)
```

Преимущество такой схемы: на сайте калькулятор работает **без интернета и без подключения к Excel**. Excel нужен только в момент обновления цен.
